# FUTURE_CS_01 - Vulnerability Assessment Report

## Task
Cyber Security Internship Task 1 by Future Interns

## Objective
Perform a read-only vulnerability assessment on a public website and document findings in a professional security report.

## Website Tested
http://testphp.vulnweb.com

## Scope
This assessment was limited to passive and read-only analysis only.

No exploitation, brute force, denial-of-service attacks, or unauthorized access attempts were performed.

## Tools Used
- Nmap
- OWASP ZAP (Passive Scan)
- Browser DevTools

## Assessment Areas
- Open ports and exposed services
- Security headers
- Cookie security
- Information disclosure
- General security misconfigurations

## Repository Contents
- Vulnerability Assessment Report
- Evidence screenshots
- Tool outputs
- Findings and remediation recommendations

## Disclaimer
This project was completed for educational purposes as part of the Future Interns Cyber Security Internship Program.
